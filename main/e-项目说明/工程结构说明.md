# 工程结构说明

## 项目结构

```bash
├── djj-business
│   ├── file-center -- 文件中心[9002]
│   └── user-center -- 用户中心[9001]
├── djj-config -- 配置中心
├── djj-demo -- 项目demo
├── djj-doc -- 项目资料
├── djj-gateway -- 服务网关Api一级工程
│   └── sc-gateway -- spring cloud gateway[8099]
├── djj-infrastructure -- 通用工具一级工程
│   ├── djj-auth-client-spring-boot-starter -- 封装spring security client端通用操作
│   ├── djj-common-core -- 通用核心
│   ├── djj-common-spring-boot-starter -- 封装通用逻辑操作
│   ├── djj-db-spring-boot-starter -- 封装数据库通用逻辑操作
│   ├── djj-loadbalancer-spring-boot-starter -- 封装loadbalance通用逻辑操作
│   ├── djj-log-spring-boot-starter -- 封装log通用逻辑操作
│   ├── djj-redis-spring-boot-starter -- 封装redis通用逻辑操作
│   ├── djj-sentinel-spring-boot-starter -- 封装sentinel的通用逻辑操作
│   └── djj-swagger2-spring-boot-starter -- 封装swagger通用逻辑操作
├── djj-monitor
│   ├── log-center -- 日志中心[7200]
│   └── sc-admin -- 应用监控[6500]
├── djj-register
│   └── nacos -- Nacos注册中心[8848]
├── djj-sentinel -- Sentinel服务降级、限流熔断中心[6999]
├── djj-uaa -- spring security认证中心[8077]
└── djj-web
    └── djj-back-web -- 后台前端[8066]
```
